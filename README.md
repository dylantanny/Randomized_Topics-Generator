# 日本語ライティング練習 / Japanese Writing Practice Topic Picker

A simple Python script that gives you a random daily writing topic in Japanese, tracking which ones you've already used so you don't repeat yourself until you've gone through all 50.

---

## Files

| File | Description |
|---|---|
| `main.py` | The main script |
| `topics.json` | Your topic list — hand-edit this to add or remove topics |
| `used_ids.json` | Auto-generated by the script — don't edit this manually |

---

## Requirements

- Python 3.x — download from [python.org](https://www.python.org/downloads/) if not installed
- No external libraries needed — uses only Python's standard library

---

## Setup

1. Put `main.py` and `topics.json` in the same folder
2. Open a terminal in that folder
3. Run the script

---

## How to run

**Windows**
```
python main.py
```
or if that doesn't work:
```
py main.py
```

**macOS / Linux**
```
python3 main.py
```

---

## How it works

Each time you run the script, it:
1. Loads all topics from `topics.json`
2. Checks `used_ids.json` to see which topics you've already done
3. Picks a random topic from the remaining unused ones
4. Prints the topic
5. Saves the updated used list back to `used_ids.json`

Once you've gone through all 50 topics, it notifies you and resets the pool so you can start again.

---

## Example output

```
food - What did you eat for breakfast today and why
```

---

## Adding or editing topics

Open `topics.json` and add entries following this format:

```json
{"id": 51, "main_topic": "travel", "topic": "Describe a place you want to visit"}
```

Make sure each `id` is unique. The script uses ids to track usage, so don't reuse or change existing ids or your tracking history may get confused.

---

## Resetting progress

Delete `used_ids.json` and the script will start fresh as if you've never run it before.
